+-------------------------------------------------------------------+
| Criar um repositório local                                        |
+-------------------------------------------------------------------+
git init

+-------------------------------------------------------------------+
| enviar arquivos para o git                                        |
+-------------------------------------------------------------------+
git add "nome do arquivo"  ou git add . enviar todos os arquivos

+-------------------------------------------------------------------+
| Verificar alterações do repositório                               |
+-------------------------------------------------------------------+
git status

+-------------------------------------------------------------------+
| Commitar alterações                                               |
+-------------------------------------------------------------------+
git commit -m "Descrição da alteração"
